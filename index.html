Asymmetric cryptography

Basic

  • m – raw data, c – enctypted data
  • One way function c = f(m)
  • You can calculate c if you know m and function's parameters
  • You can not calculate m if you know c in a reasonable amount of time
    key life time
  • You have function's parametersf() (Public key)
  • You have a trapdoor function (Private key)

How WebAuthn works

How WebAuthn works, again

  • Registering a WebAuthn Credential (User Registration)
    1. Hand over user identification to Relying party (web site, server)
    2. Generate new key-pair
    3. Deliver public key to the server
  • Authenticating with a WebAuthn Credential (User Authenticating)
    1. Hand over user identification to Relying party (web site, server)
    2. Find existing key-pair
    3. Create signature and deliver it to the server
